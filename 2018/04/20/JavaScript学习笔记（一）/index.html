JavaScript学习笔记（一）

  • JavaScript

数据类型

多行字符串不要用\换行，防止转义了空格难以发现。尽量使用闭合引号+，或者使用ES6的 ` 反引号。

Object

null表示空object，undefined表示空非object。

ES5以后object的最后一个key-value可以跟逗号，也就是说IE8以下不兼容。

object的key在声明是必定为string类型，若果不是的话自动转换为string。

当需要用key索引时，用object.[index]的方式索引时，index必须为字符串，因为中括号获取object的属性它并不会自动给你转成字符串。

当key为空字符串’’时，必须使用object[‘’]来获取。

object.atr的atr等价于object[‘atr’],所以在for..in中不应该用.atr变量value。

在用点访问对象的属性时，这个属性必须满足标识符的命名规则

why typeof nul === object

The data is a reference to an object. 而js 里的Null 是机器码NULL空指针, (0x00 is most platforms).所以空指针引用 加上 对象标记还是0,最终体现的类型还是object..这也就是为什么Number(null)===0吧…The history of “typeof null”2. 曾经有提案 typeof null === ‘null’.但提案被拒绝harmony:typeof_null [ES Wiki]
